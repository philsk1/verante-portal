## DB CHANGES MADE THIS SESSION (management API — architecture only)

```sql
-- Fixed is_member to SECURITY DEFINER:
CREATE OR REPLACE FUNCTION public.is_member(tenant uuid)
RETURNS boolean LANGUAGE sql STABLE SECURITY DEFINER SET search_path = public
AS $$ SELECT EXISTS (SELECT 1 FROM tenant_memberships WHERE tenant_id = tenant AND user_id = auth.uid()); $$

-- Fixed has_admin to SECURITY DEFINER:
CREATE OR REPLACE FUNCTION public.has_admin(tenant uuid)
RETURNS boolean LANGUAGE sql STABLE SECURITY DEFINER SET search_path = public
AS $$ SELECT EXISTS (SELECT 1 FROM tenant_memberships WHERE tenant_id = tenant AND user_id = auth.uid() AND role IN ('owner', 'admin')); $$

-- New SELECT policy (breaks chicken-and-egg on tenant INSERT):
CREATE POLICY "tenants_creator_email_read" ON tenants
FOR SELECT USING ((auth.jwt() ->> 'email') = business_email)
```
